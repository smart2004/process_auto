### Description

##### Big_M
```
Пример кода Python для создания логических ограничений с использованием Big-M подхода и бинарных переменных.
```
##### Process auto
```
Этот код позволяет загрузить данные из файла "data.csv", преобразовать дату в формат datetime, 
отфильтровать данные по определенным условиям, сгруппировать их по дате и сохранить результат в файл "result.csv". 
Таким образом, данный код может быть использован для автоматизации бизнес-процессов, 
связанных с анализом данных и отчетностью.
```
##### gdx_API
```
Пример кода API python для работы с механизмом GAMS Data Exchange (gdx).
```
